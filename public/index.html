🌿 MiniFarm Monitor

Surveillez les mesures de vos capteurs en temps réel.
Cette application simule une mini-ferme connectée avec des capteurs (ESP32 simulé).

📊 Données Temps Réel

Aucune donnée reçue.

📚 Concepts pédagogiques

  • 🔌 Simulation d'un système embarqué (ESP32)
  • 📡 Transmission HTTP vers un serveur Node.js
  • 🧠 Traitement en temps réel via WebSocket
  • 🗂️ Historique temporaire en mémoire (10 dernières mesures)
  • 🧪 Simulateur intégré pour tests automatiques

🔧 Tester l'API

Exemple de requête curl :

curl -X POST http://localhost:3000/api/data \\
  -H "Content-Type: application/json" \\
  -d '{"capteur":"ESP32", "type":"Température", "valeur":28.4}'
